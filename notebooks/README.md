# Music4All-Onion

Music4All-Onion is a large-scale, multi-modal music dataset that
expands the Music4All dataset by including 26 additional audio, video, and metadata features for 109,269 music pieces
and provides a set of 252,984,396 listening records of 119,140 users, extracted from the online music platform Last.fm .
